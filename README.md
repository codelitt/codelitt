codelitt
========

Repo for http://www.codelitt.com

Colour for brand: 0093D0

## Installation

- Create the codelitt gemset in order te use the .rvmrc:
      rvm 2.1.1@codelitt --create
- While inside the codelitt folder execute:
      bundle install
- Start the server with:
      ruby codelitt.rb
- Enjoy
